How'd you get here?

Welcome. You've somehow -- either by accident, or because I linked you -- stumbled upon my site, Tacos N' Security. Unforunately, InfosecNoob was already taken, so I had to come up with something else, and I like tacos. It's a simple site that started out as a small project to help me learn a bit about HTML5 and CSS while taking classes at WGU, but I figured it was worth building upon and maintaining. This site will soon be populated with more things as I start building it and learning more HTML.

It's a bit experimental. I'm not a web designer, afterall. I just want to learn how things work, and it makes for a good opportunity for me to build something I can use later if some employer decides they want to give me a shot. At least they can see I have some level of skill.

This is mostly going to be a collection of some things I learn and want to share or focus on. I will only update when I can, and I'll say that's not very often when you're a student and work full-time. Either way, this can be a wonderful learning experience, so I hope to learn something new.

Some background

I'm a cybersecurity student at Western Governor's University. In fact, I just completed my first year on January 1st, 2018. It's not a big deal to some, but I'm glad I've gone this far, and that I'm sticking to it.

I've been playing with computers for as long as I can remember, but I only recently started doing it as a job. I've learned a lot in the last three years, and hope to learn plenty more. In fact, I built this site to learn some basic HTML and CSS.

Above, you will find links to my blog, my resume, some cool links and some of my other interests. They are all worth checking out.
